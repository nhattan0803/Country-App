This is a Next.js project bootstrapped with create-next-app.

# Getting Started
First, set up enviroment:

- Nextjs
- Typescript
- Tailwind
- Framer motion

And, run the development server:

npm run dev
# or
yarn dev
Open http://localhost:3000 with your browser to see the result.
